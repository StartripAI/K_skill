# K.skill Promotion Copy

## English HN Copy

Show HN: K.skill - local voice-first persona packs from chats, memories, and characters

Some people and characters do not fit into a short prompt. Their voice has pauses, their chats have private jokes, and photos or stickers can hold the mood of a moment better than a clean summary.

K.skill turns voice notes, chat logs, screenshots, photos, stickers, world notes, public writing, and personal notes into local persona packs you can reopen, inspect, keep talking with, and export. A pack can carry transcript evidence, voice DNA, shared memories, character rules, confidence notes, and a Prompt Stack that shows why the persona speaks the way it does.

The product has five clear boundaries:

- Voice Memory keeps transcript evidence, voice DNA, and TTS previews inside the pack.
- Crush Coach reads a DM thread and drafts replies that still sound like you.
- Relationship Memory turns shared context into long-term memory.
- Character World builds OCs, movie characters, NPCs, lorebooks, and virtual personas.
- Life Mentor turns public material and personal notes into a thinking companion with evidence and confidence visible.

It includes a local GUI and CLI, multimodal import, local ASR / TTS hooks, Prompt Stack inspection, report downloads, memory inspection, and persona-format exports.

## 中文小红书文案

有些人不是一句“他很温柔”就能留下来的。真正会突然想起的，是语音里那半秒停顿、聊天里的老梗、照片里没说出口的表情、表情包里的默契，还有一起走过的那段路。梦里出现的人、脑内的 OC、想长期维护的虚拟人格，也常常差这一点“声音”和“共同经历”。🎧

K.skill 做的就是把这些散落的东西收成一份本地 persona pack。你可以放进 voice note、聊天记录、截图、照片、sticker、PDF、视频字幕、角色设定、世界观、公开文章和自己的笔记。它会先把材料变成能检查的 evidence，再整理出关系记忆、聊天节奏、voice DNA、角色语气、lorebook 和 Prompt Stack。不是把资料揉成一段简介，而是让你能回看：这句话从哪里来，为什么这么判断，置信度有多少。

我最喜欢它的几个入口：Voice Memory 把声音变成 transcript、voice DNA 和可预览的语气；Crush Coach 适合“TA 刚回我了，我该怎么接”的时刻，会读热度、风险和下一步；Relationship Memory 像一本会更新的关系相册，把称呼、共同经历和细小习惯留住；Character World 可以给原创角色、电影人物、NPC、虚拟人格补上身份、世界观和记忆触发；Life Mentor 则把公开资料、访谈、原则和笔记整理成可以追问的思考陪跑。

它更像一个可以慢慢补充的私人记忆空间。今天先放一条语音，明天补几张截图，后天把角色或关系导出成常用人格包格式。每一步都留着来源、evidence 和 confidence。适合整理一段关系、做一个会说话的角色，也适合把自己的思考方式沉淀下来。✨
