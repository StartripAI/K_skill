# K.skill Promotion Copy

## GitHub About

Local voice-first persona packs from the voices, chats, photos, stickers, characters, and shared memories you want to reopen.

## English HN Copy

Show HN: K.skill - local voice-first persona packs from voices, chats, and memories

Some people and characters disappear when you reduce them to a prompt. Their voice has pauses. Their chats carry private jokes. A photo, sticker, screenshot, scene card, or half-remembered note can hold the part a clean summary loses.

K.skill turns that material into a local persona pack you can reopen. A pack can carry transcript evidence, voice DNA, shared memories, character rules, source notes, confidence, and a Prompt Stack, so the persona does not come from a black box. You can see what it remembers, why it speaks that way, and where stronger claims came from.

It has five paths:

- Voice Memory keeps voice notes, transcript evidence, voice DNA, and previewable tone inside the pack.
- Crush Coach reads the temperature of a DM thread and drafts replies that still sound like you.
- Relationship Memory turns chats, photos, voice notes, and corrections into a relationship archive you can keep updating.
- Character World gives OCs, movie characters, NPCs, and virtual personas identity, world rules, memory triggers, and lore.
- Life Mentor turns public material and personal notes into a thinking companion whose evidence and confidence stay visible.

You can start from the local GUI, use the CLI for repeatable flows, connect local ASR / TTS when you are ready, inspect the Prompt Stack, and export the same pack into common persona formats.

## 中文小红书文案

有些人不是一句“他很温柔”就能留下来的。真正把你拉回去的，是语音里那半秒停顿、聊天里只有你们懂的梗、照片里没说出口的表情、表情包里的默契，还有那段一起走过的路。梦里出现的人、脑内的 OC、想长期维护的虚拟人格，也常常差一点声音和共同经历。🎧

K.skill 想做的就是把这些散落的东西收进一份本地 persona pack。你可以放 voice note、聊天记录、截图、照片、sticker、PDF、视频字幕、角色设定、世界观、公开文章和自己的笔记。它先把材料变成能回看的 evidence，再整理出关系记忆、聊天节奏、voice DNA、角色语气、lorebook 和 Prompt Stack。不是把一堆资料压成简介，而是让你能追问：这句话从哪里来？为什么是这个判断？置信度有多少？

我最喜欢的入口是 Voice Memory：一段声音会变成 transcript、voice DNA 和可预览的语气。Crush Coach 适合“TA 刚回我了，我该怎么接”的瞬间；Relationship Memory 像一本会更新的关系相册；Character World 可以让原创角色、电影人物、NPC、虚拟人格拥有身份、世界和记忆触发；Life Mentor 则把公开资料、访谈、原则和笔记整理成可以追问的思考陪跑。

它像一个慢慢补齐的私人记忆空间。今天放一条语音，明天补几张截图，后天把角色或关系导出成常用人格包格式。适合整理一段关系、让角色开口，也适合把自己的思考方式沉淀下来。✨
