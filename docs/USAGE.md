# K.skill Usage Guide

## GUI Flow

1. Run `npm run dev`.
2. Open the local URL.
3. Choose one workflow on the left.
4. Upload or paste source material.
5. Inspect stage, evidence, memory, prompt stack, and reply suggestions.
6. Save generated reports or switch to CLI for filesystem exports.

## Supported Inputs

- `.txt`: chat logs, notes, scripts
- `.md`: character/world/mentor notes
- `.json`: structured chat logs or character cards
- `.csv`: speaker/message exports
- `.html`: web chat exports

## Supported Exports

- Codex skill folder
- Claude Code skill folder
- ChatGPT instructions and knowledge bundle
- DeepSeek/OpenAI-compatible system prompt JSON
- SillyTavern Character Card V2 and lorebook
- Hermes `SOUL.md`
- LobeChat/Open WebUI agent JSON
